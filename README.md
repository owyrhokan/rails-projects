# rails-projects

Learning project for rails-projects.

## Features
- Basic CRUD operations
- User authentication
- Database management

## Tech Stack
- Ruby on Rails
- PostgreSQL

## Usage
1. Clone the repo
2. Run `bundle install`
3. Setup the database with `rails db:create`
4. Start the server with `rails server`
